# Mechanism models: subject random intercepts for repeated measures

> [!WARNING]
> These notes were generated by an AI tool.

Date: 2026-04-18

## Decision

`build_mechanism_model` gains two parameters:

- `use_subject_random_intercept: bool = True` — default on. Adds a
  non-centred child-level random intercept
  `u_child = sigma_child * u_child_raw`, with
  `u_child_raw ~ Normal(0, 1, dims="child")` and
  `sigma_child ~ HalfNormal(sigma_child_prior_sigma)`. Indexed into
  the linear predictor via `prepared.child_idx`.
- `sigma_child_prior_sigma: float = 0.5` — the prior scale on
  `sigma_child`.

LRP56 / LRP57 / LRP58 specs explicitly set the flag on.

## Why

Each mechanism fit pools 157 observations across three phase
transitions (t1→t2, t2→t3, t3→t4) for up to 53 children. Rows from
the same child share all baseline characteristics, so they are **not
independent**. Without a child-level random intercept, the model
treats 157 correlated observations as 157 exchangeable ones, which:

- overstates the effective sample size,
- anti-conservatively narrows credible intervals on β_G, γ's and
  `f_mech`,
- can load genuine between-child variability onto the mechanism
  slope rather than into a child intercept.

This is the textbook repeated-measures fix. Random intercepts for
the ITT models (LRP52/53/54) are *not* added because those fits have
one row per child — no repeated measures.

## Interpretation

1. **σ_child is small** across all three models: posterior means
   0.07–0.08, 95 % CIs reaching zero. Residual within-child
   correlation after conditioning on γ_own is modest.
2. **CI widening is small** (~5–10 % per parameter). γ_own (0.72–0.85
   across models) absorbs most of the baseline-level signal per
   child, so the remaining between-child variance that the intercept
   can absorb is limited.
3. **No estimand crosses a credibility boundary** as a result of the
   change. LRP58 L→W remains credibly positive on both amplitude
   (`f_mech__eta` lower bound 0.083) and mean slope (+0.014, +0.321).
4. **Divergence rates crept up slightly** in LRP56 and LRP57. With
   σ_child 95 % CI reaching zero, the usual σ → 0 funnel reappears on
   the child block. Still below 1 % in each — acceptable.
5. **LRP57 shows the softest sampling** (R̂ = 1.005 for alpha,
   ESS_bulk ≈ 1 400 for alpha). ESS_bulk on τ and γ_own still in the
   thousands; R̂ on the key estimands is 1.000. If a follow-up fit
   shows trouble, tightening `sigma_child_prior_sigma` from 0.5 to
   ~0.25 should remove the funnel neck since σ_child's posterior sits
   well below 0.2 anyway. Left at 0.5 so the prior does not dominate.

## LRP55 follow-up (deferred)

LRP55 has a similar repeated-measures structure — each child
contributes 8 rows (one per outcome) in the RCT phase. A shared
child-level random intercept across outcomes would absorb "overall
ability" differences that are currently factored only through γ_own
of each outcome. Conceptually distinct from the 8-dimensional LKJ
residual we dropped earlier: the random intercept is a single
scalar per child that shifts *all* outcomes in tandem, rather than
allowing independent per-outcome perturbations with a free
correlation structure.

Priority: lower than the mechanism-model change because the joint τ
posterior already matches the LRP52/53/54 univariates to three
decimal places, suggesting the shared across-outcome structure is
not mis-specified for inference on τ. Will revisit if the LRP55
posterior-predictive check reveals tension.
