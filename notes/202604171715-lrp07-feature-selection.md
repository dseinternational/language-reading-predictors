## Motivation

LRP07 predicts receptive vocabulary **gains** (`rowpvt_gain`) —
paralleling LRP01 (word-reading gains), LRP03 (expressive-vocabulary
gains), and LRP05 (letter-sound knowledge gains). Baseline started
from the full `Predictors.DEFAULT_GAIN` set plus the auto-included
`rowpvt` base variable (34 predictors total), n=161, with no
outlier exclusion.

Following the established playbook, MAE hyperparameter tuning on
the full predictor set precedes any feature-selection work.

- **Base `rowpvt` dominates** at 0.286 but much less than at
  baseline (1.274). Tuning redistributed signal massively away

  but more extreme here.

  The first predictor beyond the base level is grammar, not
  vocabulary — interesting given the target is a vocabulary gain.

  appears as meaningful, consistent with LRP01 / LRP05 findings.
- **`time` at 0 importance** — unusual. In LRP01 / LRP03 / LRP05
  the time-between-timepoints was a strong predictor of gains.
  For rowpvt_gain, the 42-tree shallow-depth model apparently
  doesn't benefit from it — possibly saturated by rowpvt itself
  or because the gain is less time-dependent than other gains.
- **`eowpvt` at 0 importance** — striking. Expressive vocabulary
