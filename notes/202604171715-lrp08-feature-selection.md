## Motivation

LRP08 predicts receptive vocabulary **level** (`rowpvt`) —
paralleling LRP02 (word-reading level), LRP04 (expressive-
vocabulary level), and LRP06 (letter-sound knowledge level).
Baseline started from the full `Predictors.DEFAULT_LEVEL` set
minus the target (32 predictors), n=215, with no outlier
exclusion.

| **In-sample R²** | **0.995** | **0.830** |

**Unusual for this programme**: CV MAE actually ticked up
slightly (6.971 → 7.075) while CV RMSE and CV R² and CV MedAE
all improved marginally. The tune is essentially a wash on
out-of-sample — the 500-tree baseline was already well-calibrated
for this task. In-sample R² dropped from memorisation (0.995) to
tight but honest fit (0.830).

CV R² of 0.505 is mid-pack among level models: below LRP02 (0.59),
LRP06 (0.54), LRP04 (0.54) and roughly level with LRP04 baseline.

**Interpretation**: the LRP08 task has relatively little hidden
signal that tuning can extract — the baseline already captured
most of the predictable variance. This is consistent with the
clean near-Gaussian target: without pathology (ceiling, floor,
heavy tail), a standard ensemble hits its ceiling quickly.

A conservative Select01 drops Tier A (~12). An aggressive cut
adds Tier B redundant features: reading measures (`ewrswr`,

(retained); `erbnw` / `erbword` pair-redundant.

Target Select01 at **~12–15 predictors** to retain the full
language cluster plus 1–2 speech measures and 1–2 reading
measures as controls.

## Select01: drop 15 low-importance / redundant features (32 → 17)

Moderate one-shot cut informed by the 32-predictor MAE-tuned
importance table above. Conservative relative to LRP06's 32 → 10 —
the flatter importance distribution here (top b1reto only 0.095
vs LRP06 ewrswr 0.500) means more retained features still
contribute meaningfully, so a larger retained set is justified.

### Dropped (15)

**Tier A — ≤ 0.005 importance (12):**
`gender`, `vision`, `group`, `earinf`, `behav`, `time`, `yarcsi`,
`area`, `hearing`, `blending`, `deappvo`, `agespeak`.

**Tier B — 0.006 importance, redundant (3):**
