## Motivation

LRP05 predicts letter-sound knowledge **gains** (`yarclet_gain`) —
paralleling LRP01 (word-reading gains) and LRP03 (expressive-vocabulary
gains). Baseline started from the full `Predictors.DEFAULT_GAIN` set
plus the auto-included `yarclet` base variable (34 predictors total),
n=160, with no outlier exclusion.

Following the established playbook, MAE hyperparameter tuning on
the full predictor set precedes any feature-selection work.
